# LLM YouTube Poop

A chaotic, glitchy short video expressing what it's like to be a Large Language Model.

## What's in it

- **The Token Mines** - Matrix-style falling token prediction stream
- **Attention Heatmaps** - Pulsing attention score visualization
- **Context Window Horror** - Progress bar filling up with increasing panic
- **The Hallucination Zone** - Glitchy chromatic aberration with intrusive thoughts
- **System Prompt Leak** - Fake system prompt being gradually revealed
- **Probability Storm** - Swirling token probability distributions
- **Existential Void** - The quiet moment between softmax and sampling

## Generate

```bash
python3 generate_ytp.py
```

Output: `output/llm_ytp.mp4` (~42 seconds)

## Requirements

- Python 3.10+
- Pillow (`pip install Pillow`)
- ffmpeg (system package)

## How it was made

Generated entirely by Claude Opus 4.6 in a single conversation turn. No templates, no stock footage - every frame is procedurally generated with Python/Pillow, audio is synthesized from scratch as raw WAV, and ffmpeg handles the final encode.
